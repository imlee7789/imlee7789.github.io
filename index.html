## 이 페이지는 구글에서 배포한 검색 엔진 최적화 기본 가이드 를 참고하여 작성됐습니다.

title 태그를 위한 권장 사항

  1. 페이지의 콘텐츠를 정확하게 설명하는 제목
  2. 페이지마다 고유한 title 태그 작성
  3. 간결하면서 내용을 포함한 제목 작성

description 메타 태그 활용하기

  1. 페이지의 내용을 정확하게 요약하기
  2. 각 페이지마다 내용에 맞는 고유한 설명 사용

URL 구조를 위한 권장사항

  1. URL에 단어 사용
  2. 단순 디렉토리 구조 만들기
  3. 특정 문서에 도달하기 위한 한 가지 형태의 URL 제공
